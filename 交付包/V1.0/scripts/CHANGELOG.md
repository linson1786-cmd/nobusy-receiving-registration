# 收货信息登记 Skill 变更日志

## V1.0 - 2026-06-23

### 初始发布

- **交互式信息采集**：通过对话逐步采集收货公共信息和物品明细，支持单物品/多物品批量登记
- **智能字段校验**：仪器设备类序列号必填、批次号选填；其他类型批次号必填、序列号选填
- **数量校验**：数量必须为正数，兼容字符串传入
- **Excel 表初始化**：首次使用时创建标准化收货登记表（18列表头、冻结首行、自动筛选、列宽样式）
- **自动编号**：RG-YYYYMMDD-NNN 格式，当日序号从001递增，多物品自动连续编号
- **记录查询**：按编号精确查询，按日期/供应商模糊查询
- **记录更新**：按编号定位更新，更新后保留单元格样式
- **主体隔离**：各业务主体独立 Excel 文件，互不混用
- **业务主体注册引导**：首次使用时引导用户补充业务主体信息
- **版本管理**：Git 模式版本控制，deploy.py 自动部署，更新不影响用户数据
