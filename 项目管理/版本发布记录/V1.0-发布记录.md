## 发布概述

收货信息登记 Skill 的初始发布版本，包含完整的交互式收货信息采集、Excel 写入、自动编号、记录查询与更新功能。由 Skill獭造团队（潜溪-需求分析、汀匠-结构设计、澄鉴-质量校验）全流程制作完成。

## 使用人指引

### 首次使用

1. 安装依赖：`pip install openpyxl`
2. 运行部署：`cd scripts && python3 deploy.py`
3. 在 WorkBuddy 对话中说"初始化收货表"
4. 按引导提供主体名称和保存路径
5. 说"收货登记"开始使用

### 更新指引

```bash
cd scripts/receiving-registration
python3 deploy.py
```

部署工具自动检测版本，有新版本时备份旧文件并更新。

## 版本兼容性

- **向前兼容**：V1.0 数据可在后续版本继续使用
- **向后兼容**：不支持降级到 V1.0 以下（无更低版本）
- **数据安全**：更新只更新 Skill 文件，不触碰用户数据（~/收货登记/*.xlsx）
