## Excel 文件存储规范

### 默认存储路径

```
~/收货登记/收货登记表_{主体简称}.xlsx
```

### 示例

```
~/收货登记/收货登记表_NoBusy.xlsx
~/收货登记/收货登记表_NoBusy-Demo.xlsx
```

### 使用流程

1. 首次使用时，用户告知项目归属
2. 系统根据主体简称生成默认文件路径
3. 若文件不存在，提示用户初始化（流程B）
4. 后续登记直接使用已有文件

## 主体选择交互规则

- 用户说出主体名称（全称或简称均可）时，匹配上表对应行
- 匹配不到时，列出所有可选主体供用户选择
- 用户未指定时，主动询问"本次收货属于哪个主体？"
- 若主体列表中有未填写的占位项（含方括号），仅展示已填写的主体供选择，并提示用户可补充其他主体名称
